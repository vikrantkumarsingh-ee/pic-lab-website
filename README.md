# PIC Lab website

Static website for the Precision Instrumentation and Control Laboratory, Department of Electrical Engineering, IIT Kharagpur.

## Local preview

Open `index.html` in a browser. No installation or build step is required.

## Main pages

- Home
- Current Research and three detailed current-research pages
- Selected Previous Research and five detailed project pages
- People
- Publications
- News
- Opportunities
- Contact

## Updating content

Edit the HTML files in a text editor. Images are stored under `assets/images/`; videos are stored under `assets/videos/`.

## Publishing

See `GITHUB-PAGES-GUIDE.md` for browser-only instructions aimed at a first-time GitHub user.
